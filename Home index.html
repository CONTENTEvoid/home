Sobre o Projeto

O Stream Vision é um projeto desenvolvido para a disciplina de desenvolvimento web. Seu propósito é demonstrar como construir um site simples para uma plataforma de TV por IP, de forma legal e educativa, utilizando tecnologias como HTML, CSS e JavaScript.
